# Skleppie (WORK IN PROGRESS)
Simple e-commerce shop with Spring Boot and MySQL (CRUD app).

## Technologies I used

* Java SE 8
* Spring Boot
* Spring Security
* MySQL
* JPA and Hibernate
* TomCat
* Maven
* HTML and CSS (Bootstrap)
* A little of JavaScript
